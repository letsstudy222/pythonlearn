S.6 — Món bán chạy nhất MỖI danh mục
    
Câu hỏi phỏng vấn kinh điển "top 1 mỗi nhóm": với mỗi danh mục, món nào có tổng số lượng bán cao nhất? (Gồm 3 tầng: JOIN + GROUP BY → ROW_NUMBER PARTITION BY → lọc hang = 1.)

    
st_6.sql
Ctrl + Enter

    
Gợi ý
CTE 1: tổng số lượng theo (danh_muc, ten). CTE 2 (hoặc lồng): thêm ROW_NUMBER() OVER (PARTITION BY danh_muc ORDER BY tong_sl DESC). Cuối: WHERE hang = 1 — mẫu chuẩn của bài S9.

    
Lời giải
WITH ban_hang AS (
  SELECT s.danh_muc, s.ten, SUM(d.so_luong) AS tong_sl
  FROM don_hang d
  JOIN san_pham s ON d.ma_sp = s.ma_sp
  GROUP BY s.danh_muc, s.ten
),
xep_hang AS (
  SELECT *,
    ROW_NUMBER() OVER (PARTITION BY danh_muc ORDER BY tong_sl DESC) AS hang
  FROM ban_hang
)
SELECT danh_muc, ten, tong_sl
FROM xep_hang
WHERE hang = 1;
