Python là một ngôn ngữ lập trình — tức là cách con người viết chỉ dẫn để máy tính thực hiện. Trong các ngôn ngữ phổ biến, Python nổi tiếng vì dễ đọc nhất: code Python trông gần giống tiếng Anh thường.

  
Python được dùng ở khắp nơi: phân tích dữ liệu, tự động hóa công việc văn phòng, xây website (Instagram, YouTube dùng Python), và là ngôn ngữ số 1 của trí tuệ nhân tạo (AI).

Viết hàm tinh_bmi(can_nang, chieu_cao) trả về chỉ số BMI (cân nặng kg chia cho bình phương chiều cao mét). Sau đó viết hàm phan_loai(bmi) trả về "Gầy" nếu BMI < 18.5, "Bình thường" nếu < 25, còn lại "Thừa cân". Dùng cả 2 hàm với số liệu của bạn.

Gợi ý
Bình phương: chieu_cao ** 2. Hàm phân loại dùng if/elif/else, mỗi nhánh một return.

    
Lời giải
def tinh_bmi(can_nang, chieu_cao):
    return can_nang / chieu_cao ** 2

def phan_loai(bmi):
    if bmi < 18.5:
        return "Gầy"
    elif bmi < 25:
        return "Bình thường"
    else:
        return "Thừa cân"

bmi = tinh_bmi(62, 1.68)
print(f"BMI = {bmi:.1f} → {phan_loai(bmi)}")

Tự viết "Sổ chi tiêu mini": tạo một list rỗng, dùng vòng while hỏi người dùng nhập số tiền từng khoản chi (nhập 0 để dừng), append vào list. Kết thúc: in số khoản chi, tổng chi, khoản lớn nhất. Viết hoàn toàn tự do — bí mới xem lời giải.

Gợi ý
Khung: chi_tieu = [] → while True: → nhập tiền → nếu tien == 0: break → ngược lại chi_tieu.append(tien). Cuối cùng dùng len(), sum(), max().

    
Lời giải
chi_tieu = []
print("Nhập từng khoản chi, nhập 0 để kết thúc")
while True:
    tien = float(input("Khoản chi (đ): "))
    if tien == 0:
        break
    chi_tieu.append(tien)

if len(chi_tieu) > 0:
    print(f"Số khoản chi: {len(chi_tieu)}")
    print(f"Tổng chi: {sum(chi_tieu):,.0f} đ")
    print(f"Khoản lớn nhất: {max(chi_tieu):,.0f} đ")
else:
    print("Bạn chưa nhập khoản nào.")
